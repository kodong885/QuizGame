# 마리오 ppt 모둠 게임
우리가 어릴적 학교 수업시간에 모둠을 정해서 재미있게 했었던 마리오 ppt게임을 만들어보았다. 
프레임워크는 사용하지 않았고, 단순히 자바만을 사용하여 개발하였다. 

## 게임 설명
그룹들이 순서대로 A 부터 Z까지의 퀴즈를 선택해서 O, X로 퀴즈를 풀이함.
만약 해당 그룹이 퀴즈를 맞출시, 10점을 얻고, 맞추지 못하면 아무런 점수가 없음.
이후, 선택된 퀴즈 다시 선택할 수 없고(사라짐), 그룹이 선택할 수 있는 퀴즈가 없으면 게임이 끝남.

게임이 끝남과 동시에 최다 득점자(중복 일 수도 있음)를 축하해주며 게임종료.
